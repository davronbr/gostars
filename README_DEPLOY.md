# Botni faollashtirish (ENG OSON YO'L)

### 1. GitHub-ga kodni yuklash (DIQQAT!)
Siz GitHub-ga kod yuklashda **Telegram Bot Tokenini** (871120... bilan boshlanadigan) ishlatmang. GitHub bunga ruxsat bermaydi.

**Sizga GitHub Token kerak (ghp_... bilan boshlanadi).**

**To'g'ri buyruq (O'zingizning ghp_... TOKENingizni yozing):**
```bash
git remote set-url origin https://TOKENINGIZNI_SHU_YERGA_YOZING@github.com/davronbr/gostars.git
git add .
git commit -m "update bot"
git push
```

### 2. Botni bir marta faollashtirish
Saytingiz Render-da yonganidan so'ng (Build success):
1. Ilovangizga brauzerda kiring (`gostars.onrender.com`).
2. **Sozlamalar** (Settings - Tishli g'ildirak belgisi) bo'limiga kiring.
3. **"Botni faollashtirish"** (Activate Bot) tugmasini bosing.

**Tamom!** Endi botingiz Telegram-da xabarlarga javob berishni boshlaydi.

**Eslatma:** Agar bot javob bermasa, avval saytingizga brauzerda kiring, u "uyg'onadi" va bot ham ishlay boshlaydi.
