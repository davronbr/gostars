# **App Name**: Build IO

## Core Features:

- User Authentication & Profiles: Secure user registration and login, with distinct roles for buyers and developers. Users can manage their basic profile information.
- Website Marketplace: Browse and filter a list of websites available for sale, view details (name, price, screenshots, technologies, demo link), and initiate purchases. Features search and filtering by price, technology, and type.
- Developer Directory: Discover and connect with site developers. Each developer will have a profile card displaying their name, rating, specialization, and a 'Contact' button.
- Website Listing Form: A comprehensive form allowing users to list new websites for sale, including fields for name, description, price, multiple image uploads, and multi-select for technologies used. Form will include validation.
- Developer Portfolio Management: Developers can update their professional profiles, including experience, rates, and display their completed works and portfolios.
- Order & Transaction Tracking: System to manage orders, including who bought what and the current status of the transaction for both website purchases and service engagements.
- AI-Powered Listing Assistant Tool: A tool that uses AI to suggest or refine compelling descriptions for website listings based on provided technical details and features, helping sellers highlight key benefits.

## Style Guidelines:

- Background color: A premium dark interface, utilizing a subtle bluish-grey (#191D24) for depth and sophistication in Dark Mode.
- Accent color: Neon blue for all interactive elements, calls-to-action, highlights, and price tags, providing a futuristic and vibrant contrast.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and strong visual impact.
- Body text will use 'Inter' (sans-serif) for its excellent readability and neutral, objective appearance, suitable for content-heavy sections.
- Employ clean, minimalist outline icons that complement the modern, dark aesthetic, focusing on symbols related to development, commerce, services, and user actions.
- Product cards will incorporate stylized 3D illustrations to enhance visual appeal.
- A responsive, grid-based layout optimized for showcasing digital assets and developer profiles in an organized and visually appealing manner.
- Prominent search bar and filter tags at the top of content-heavy pages (e.g., Marketplace) for efficient navigation.
- Product cards featuring 3D illustrations and distinct price tags presented within blue, pill-shaped buttons.
- A floating, frosted glass bottom navigation bar for primary app navigation, inspired by the Telegram Mini App aesthetic.
- Clear hero sections on landing pages and dedicated filter/search sidebars to enhance usability on listing pages.
- Subtle and fluid micro-interactions, such as card hover effects, smooth transitions during filtering or sorting actions, and feedback animations for form submissions, to create a refined and engaging user experience.
- Overall aesthetic: High resolution, sleek glassmorphism effects, futuristic feel, and clean professional UI/UX, designed for a modern 4k display environment.
